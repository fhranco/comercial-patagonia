# 🏗️ Comercial de la Patagonia | Sistema Híbrido Pro

Este es el frontend de alto estándar para **Comercial de la Patagonia**, construido con **Next.js (App Router), TypeScript y React 19**.

## 🚀 Arquitectura Híbrida (Headless)
- **Frontend:** Next.js @ latest (TypeScript).
- **Backend:** WooCommerce REST API (@darkorange-bat-658298).
- **Estilo:** Custom Design System con CSS Modules & CSS Variables (Performance enfocada).

## 🎨 Identidad Visual (Basada en el Backend)
- **Primario:** Royal Blue (#0047AB) - Tomado del logo.
- **Acento:** Golden Yellow (#FFC300) - Tomado de la casa del logo.
- **Neutrales:** Slate Gray (#2F4F4F) para textos, White (#FFFFFF) para limpieza.
- **Tipografía sugerida:** Inter / Montserrat.

## 📁 Estructura Pro
- `src/app/`: Rutas y layouts.
- `src/components/`: Componentes atómicos (Button, Card, Grid).
- `src/lib/`: Utilidades y cliente de WooCommerce API.
- `src/types/`: Definiciones de interfaces para TypeScript.
- `src/styles/`: Tokens de diseño y globales.
